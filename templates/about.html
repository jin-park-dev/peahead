About


Data Dash Board is a personal dashboard where all my personal data are collected from various sources.
These sources range from apps on the phone, fitbit, pedometer, excel, CSV files.

They're collected and processed using python module pandas. Data are clean up to be uniform format suitable for database and depending on data average per day is calculated and stored.
Further more taking day a unit django ORM is used get data user chooses. For example Day, week, month, year etc.
